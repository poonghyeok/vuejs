디렉티브의 역할은 표현식의 값이 변경될 때 발생하는 부수효과들을 반응적으로 DOM에 적용하는 것이다.

v-on 과 v-for를 제외하고 v- 가 붙은 것들을 디렉티브라고 한다.

v-bind directive는 반응적으로 HTML 속성( id, class 등 )을 갱신하는데 사용한다.

is show : {{isShow}} span의 클래스를 boolean state를 통해 갱신할 수 있다.

DOM 이벤트를 수신하는 on directive도 있다.

동적 전달인자 v-on:[eventName]과 같이 eventName에 대한 값을 동적으로 전달하고 이벤트 핸들린을 할 수 있다.
단 eventname과 같은 것은 인스턴스를 정의할 때 들어가야한다.
